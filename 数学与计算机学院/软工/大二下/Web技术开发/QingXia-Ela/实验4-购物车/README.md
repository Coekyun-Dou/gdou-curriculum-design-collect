# 购物车实验

这个实验相对来说比较麻烦，因为要用到 javaweb 相关技术。我在这里为了方便就直接使用 springboot + vue3 了。

## 启动项目

### 前端

使用 nodejs + vue3 开发

```bash
cd src/main/resources/frontend
npm install
npm run dev
```

### 后端

使用 springboot 开发

仓库本人没有试过在命令行单独启动

用的是 idea 自带的生成 springboot 项目配置

如果有佬懂的话请务必帮忙写写

启动类是 `src/main/java/com/lab4/lab4springboot/Lab4SpringBootApplication.java`
