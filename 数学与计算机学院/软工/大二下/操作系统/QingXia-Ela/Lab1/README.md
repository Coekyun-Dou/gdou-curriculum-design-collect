# Lab1: 系统调用

## 如何启动

将本文件夹内除 readme 的所有文件替换到 xv6 仓库内，然后 `make qemu` 即可

```bash
$ hello
```

### 系统调用

```bash

$ cat [filename] # 读取指定文件并输出内容到控制台
```
